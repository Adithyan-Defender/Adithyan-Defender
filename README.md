## About

I build **secure AI-powered systems** for threat detection, vulnerability analysis, and intelligent automation.  
My work combines **LLMs, penetration testing, and secure backend engineering** to design systems that are **intelligent, resilient, and auditable**.

**Core interests**
- LLM Security, prompt injection testing, agent & tool security
- Retrieval-Augmented Generation (RAG) with secure data boundaries
- OWASP-based Web/API security and secure architecture
- SIEM-style monitoring, detection engineering, and incident-ready systems

## Selected Work

### AI Pentest / LLM Security Model
LLM-powered system for **automated penetration testing** with RAG-assisted context building and vulnerability reasoning.

- Contextual analysis of targets & attack surface
- Structured findings + reproducible evidence
- Security-first orchestration (validation, tool constraints)

### Advanced AI Voice Assistant (Secure Backend)
A secure FastAPI backend with **LLM orchestration**, validation layers, and modular services.

- Input validation and policy enforcement
- Clear separation of tools/permissions
- Observability for traces, logs, and auditing
